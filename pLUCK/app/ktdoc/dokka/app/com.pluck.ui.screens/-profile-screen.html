Displays the profile management screen allowing entrants to review and update their information.

Parameters

userName

Current display name for the entrant.

userEmail

Optional email address tied to the profile.

userPhone

Optional phone number tied to the profile.

profileImageUrl

Optional Cloudinary URL for the entrant's profile photo.

deviceId

Unique device identifier used as the Cloudinary public ID.

isLoading

Whether the overall profile data is loading.

isUpdatingProfile

Whether profile details are currently being saved.

isAdmin

Whether the current device has admin privileges.

updateMessage

Optional success banner message for profile operations.

updateError

Optional error banner message for profile operations.

onProfileImageUploadStarted

Callback invoked when the user selects a photo and upload begins.

onProfileImageUploaded

Callback invoked when a Cloudinary upload succeeds with the secure URL.

onProfileImageUploadFailed

Callback invoked when the upload fails with a human-readable message.

onUpdateProfile

Invoked when profile text fields are saved.

onSignOut

Invoked when the user taps the sign-out action.

onDeleteAccount

Invoked when the user confirms account deletion.

onMyEvents

Navigates to the "My Events" screen.

onOrganizerDashboard

Navigates to the organizer dashboard.

onAdminDashboard

Navigates to the admin dashboard (if available).

onRegisterAdmin

Invoked to open the admin registration dialog.
